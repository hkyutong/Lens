# Lens 昱镜

**面向真实生产场景的 AI 学术与创作工作台。**

Lens 昱镜把「多模型对话」「学术插件」「报告生成」「后台运营」整合到一个产品里，让团队在同一套系统里完成研究、写作、知识沉淀与交付。

## 产品定位

- 一站式：聊天、学术处理、报告导出、后台管理在同一平台完成
- 可运营：支持用户、积分、模型、插件的完整后台管理
- 可扩展：支持接入自定义模型与学术能力链路
- 可上线：提供可执行的部署文档与生产验收脚本

## 核心能力

- 多模型对话：支持模型配置、路由与调用治理
- 学术插件：PDF/Word/LaTeX/Arxiv 等学术处理能力
- 报告体系：统一品牌化导出（Lens Report｜昱镜报告）
- 权限与安全：登录鉴权、管理员能力、接口权限隔离
- 计费能力：积分扣费、余额校验、并发扣费一致性

## 适用场景

- 学术论文阅读、总结、翻译、润色
- 团队知识库内容生产与结构化沉淀
- 企业 AI 助手平台的私有化部署
- 需要后台运营能力的 AI 产品化项目

## 项目结构

- `service/`：后端 API 服务（NestJS）
- `chat/`：用户前端（Vite + Vue）
- `admin/`：管理后台（Vite + Vue）
- `academic-4.0/`：学术能力引擎（Python）
- `docs/`：部署、开发与功能文档
